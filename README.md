# 🌐 Projeto Final - Desenvolvimento Web I

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como trabalho final da disciplina de Desenvolvimento Web I do curso de Análise e Desenvolvimento de Sistemas.

O objetivo foi aplicar na prática os conhecimentos adquiridos ao longo da disciplina, envolvendo desenvolvimento de páginas web, estilização, organização de conteúdo e implementação de funcionalidades utilizando tecnologias front-end.

## 🎯 Objetivos do Projeto

- Aplicar conceitos de HTML, CSS e JavaScript;
- Desenvolver uma interface web funcional e intuitiva;
- Trabalhar com organização e estruturação de páginas;
- Utilizar boas práticas de desenvolvimento;
- Exercitar o trabalho colaborativo utilizando Git e GitHub.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS
- JavaScript
- Git
- GitHub

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados conceitos como:

- Estruturação semântica de páginas web;
- Estilização e responsividade;
- Manipulação de elementos com JavaScript;
- Organização de arquivos e código;

- Desenvolvimento em equipe.
